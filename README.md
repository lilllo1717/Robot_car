#  Robot Car Project

##  Overview  
Designing interactive robot car

##  Features  
- 4-wheel drive using gear motors  
- Obstacle detection with ultrasonic sensor  
- Motor control 
- Portable power using batteries  
- Expandable design for future features (Bluetooth, line following, etc.)
